Exam

Q1: What is the difference between `==` and `===` in JavaScript?

Answer: `==` checks for equality but allows type conversion, whereas `===` checks for both value and type without conversion.

Q2: What are closures in JavaScript?

Answer: A closure is a function that remembers the variables from its outer scope even after the outer function has finished executing.

Q3: What is the purpose of the `map()` function in JavaScript?

Answer: The `map()` function is used to create a new array by applying a given function to each element of an existing array.

Q4: How does `async/await` work in JavaScript?

Answer: `async` makes a function return a promise, and `await` pauses execution until the promise resolves, making asynchronous code easier to read.

Q5: What is the difference between `var`, `let`, and `const`?

Answer: `var` is function-scoped, `let` is block-scoped, and `const` is also block-scoped but cannot be reassigned.
